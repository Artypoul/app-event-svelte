# app-event-svelte

Базовый каркас Svelte-проекта для переноса нового дизайна.

## Что оставлено

- Минимальная структура Svelte-приложения (`src/main.js`, `src/App.svelte`).
- Базовая статика в `public/` для запуска приложения.
- Конфигурация сборки на Rollup.

## Быстрый старт

```bash
npm install
npm run dev
```

## Полезные команды

- `npm run dev` — запуск в режиме разработки.
- `npm run build` — production-сборка.
- `npm run start` — запуск собранной версии.
- `npm run check` — проверка сборки.
